PROJECT DETAILS

This project is a CSS and JavaScript-powered analog clock
designed with a structured layout, featuring a circular frame, tick marks, and animated clock hands that rotate dynamically to represent the current time.
The design is heavily CSS-driven, with properties like clip-path used to create the pointed clock hands and transform-origin ensuring accurate rotation from the center.

Key Features:

- Clock Structure:
The .con container holds the entire clock, centering it using transform: translate(-50%, -50%).
The .body serves as the clock face, enclosed within .frame-side to create the outer rim.

- Clock Hands:
The hour (.hh), minute (.mh), and second (.sh) hands are designed using clip-path polygons to create a pointed effect.
They are positioned using transform: translate(-50%, 0%) so they rotate from their top point.

- Time Markers:
The .time-con div contains tick marks (.time), which are positioned using transform: rotate(calc(30deg * var(--i)))
to distribute them evenly around the clock face.

- Animations:
Keyframes (@keyframes rotate-hour, rotate-minute, rotate-second)
ensure smooth rotation, updating the hands in real-time.

Interactive Feature:

The center button inside .centre allows interaction.
When clicked, it can display project details, explaining how the clock was made.
This part is still under development, but it will be a key interactive element.
